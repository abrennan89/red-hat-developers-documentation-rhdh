= Adding a custom application configuration file to {ocp-brand-name}
:context: assembly-add-custom-app-file-openshift

toc::[]

To access the {product}, you must add a custom application configuration file to {ocp-brand-name}. In {ocp-short}, you can use the following content as a base template to create a `ConfigMap` object named `app-config-rhdh`:

[source,yaml]
----
kind: ConfigMap
apiVersion: v1
metadata:
  name: app-config-rhdh
data:
  app-config-rhdh.yaml: |
    app:
      title: {product}
----

You can add the custom application configuration file to {ocp-short} in one of the following ways:

* The {product} Operator
* The {product} Helm chart
